## How to Setup in your local enviroment :-

### Frontend 
    1. cd frontend
    2. yarn
    3. Setup Env
    3. yarn run dev


### Blockchain
    1. cd blockchain
    2. yarn install
    3. setup env for polygon test network or you can use localhost.
    4. yarn hardhat run scripts/deploy.js --network polygon
